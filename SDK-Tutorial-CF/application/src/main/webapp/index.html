Welcome to your first web application powered by SAP S/4HANA Cloud SDK.

This file will be seen when accessing the root path of your deployed application. By default any static content inside this directory ( ./src/main/webapp/ ) is publicly routed and can be loaded by HTTP. In case you are not seeing this page via a web browser, try and locally run the project by executing the Maven command " mvn package tomee:run ".

The easiest way to get started is by navigating to your HelloWorldServlet. You will find the corresponding Java source file in this project ( ./src/main/java/ ). From here on the development will take off!

Further information and tutorials for the SAP S/4HANA Cloud SDK can be found at SAP.com. In-depth example projects can be found at GitHub.com.
